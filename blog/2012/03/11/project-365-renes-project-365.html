[Project 365] Rene's Project 365

by Rene Wang on #life, #Photos, 2012-03-11 20:51

A slightly irrelevant English version of this post is here: [Preface] About Project 365

這一切都緣起於去年的某一個無聊時刻，我上網 Google 伴我世界趴趴走的撒魯先生的資訊和照片(真的是很無聊呀!)。意外的在 Flickr 網站，看到了一個非常可愛的照相計畫，叫做 365 Saru Rururu。 這個計畫，其實就是以 Saru Rururu 為主角，也就是我的撒魯先生的英文正名，持續一年三百六十五天，每天都拍一張相片。計畫裡的撒魯先生，真是可愛的不得了。而撒魯先生的每一天，都像是有新鮮事一樣，有時認識了新的朋友，而一起留影。有時則藏匿在家裡的廚櫃裡，偷偷摸摸地找零食裹肚。喜愛攝影的我，覺得能用影像來講述每一天的故事，實在是太有趣了 !  也決定來進行自己的 Project 365計畫。

左邊就是自以為猴比花嬌的撒魯先生。

秉著增進攝影技術，並化每一日的腐朽平常事為神奇的精神，我的 Project 365 計劃將會從每日的生活中取材，以這個孕育我的城市為背景，記錄每日生活的芝麻綠豆小事。當然，經常在敝小格擔任模特兒的撒魯先生和小愣也會偶爾客串演出。我發現，要幫他們兩個想三百六十五個主題拍照實在是非常困難的事，尤其要將他們在大庭廣眾之下拿出來擺好姿勢拍照，也需要相當的勇氣。這是為什麼呢 ? 接著，就讓我的小小模特兒們，揭露一下假日時候的我，通常都在做甚麼…

事件一 :  在大庭廣眾之下指使小愣當模特兒，拍攝個人唯美沙龍照 (哪來的唯美?!)
事件二: 在大庭廣眾之下製造謎樣的命案現場?! (妳以為妳在拍攝福爾摩”愣”嗎?)
事件三: 在大庭廣眾之下製造謎樣的雙屍命案?!
為了公平起見，換個角度來一張

啊~果然是難以登入大堂的奇怪嗜好呀 ! (不過這也要感謝與我同行的 J，擔任模特兒們的技術指導呀)~所以，為了避免我的 Project 365 變成兇殺案的照片集，我還是悄悄地把我的小小模特兒們先擺在一旁，先暫時專注在發掘每天的小發現吧 !

其實，就像是每次許下的新年希望，總是希望自己能每天運動，健康又順利地瘦下幾公斤，要能持續做到每天都有主題可以拍照，想想還真是有點難度。尤其我的生活，總是順著同一條通勤路徑，來到公司，經過八小時呆滯地望著電腦之後，又順著同一條路徑回到家裡，繼續對著電視發呆。生活，對我可說是乏味之極。可是，自從我開始 Project 365 一周後 (從 3/5 開始)，我發現我開始注意身邊一些微小的事物，自己就像是一個擁有空白畫布的畫家，總是想著要怎麼把每日細心觀察到的小小驚奇，放入到畫布之中，重新創作並且加以記錄。生活，不再只是盲目的過著，而是有了全新的視野，在每日必經的軌跡中有了新的詮釋。

但是，人總是容易就懈怠的，尤其是我這個好逸享樂的金牛座。為了讓 Project 365 使命必達，除了在敝小格大聲嚷嚷我的計劃，我更另外開了一個英文的部落格 — Rene’s Proejct 365，來督促自己至少每個星期更新計劃的進度 ，而用英文的緣故，則是因為會同步在 Flickr上張貼。於是乎，我與我的 iPhone 4S 和 Sony Nex 5N，以及撒魯先生和小愣們在這個 2012 年，將要一起努力展開一趟旅程，不知道甚麼會在終點等待著我，但是期望自己能盡了全力去完成它了 !
